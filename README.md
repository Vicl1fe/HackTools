# HackTools
为了提高效率，自己写的一个小工具，来方便日常的工作。

## 功能
### 加解密
- 目前支持AES、DES、MD5(只有加密)、Weblogic、Druid、Xshell
- 加密时默认输出为Base64编码
- 解密时默认输入为Base64编码
- 密钥的编码可选择编码方式(BASE64、TEXT)
- Xshell、Druid等输入的时候一定要选择密钥编码方式为TEXT

### 杀软查询
- 集成了市面上常见的杀软和程序

### 文本处理
- 目前只支持去重



### Bug
- 给文本域加上滚动条后，UI界面会时不时的变化，所以暂时没有添加滚动条功能。
